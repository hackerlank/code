ajax组件

1、通过get的type方式进行ajax的请求：$app.ajax.get();

2、JSON方式AJAX请求：$app.ajax.getJSON();

3、AJAX跨域：$app.ajax.getJSONP();

4、以post方式进行AJAX请求：$app.ajax.post();

点击这里从后台返回hello

function goAjax(){
	loader('ajax/ajax',function(){
		$app.ajax.get('hello.html',function(e){alert(e);});
	});
}

5、同步的AJAX请求：$app.ajax.sync(url,type,data);该函数没有回调函数，直接返回后台数据。点击测试

function tAjaxSync(){
	loader('ajax/ajax',function(){
		var t = $app.ajax.sync('hello.html','get',{},'json');
		alert(t);
	});
}
