UI类Modal组件

Modal组件是弹出层组件，支持三种方式的数据源：iframe，file，id。

适用情况：多适用于弹出方式填写表单或者活动规则之类的包含页面。

关闭弹出层调用：

javascript:$app.modal.removeAll();

参数说明：

参数1：需要加载的页面地址（绝对地址和相对地址都可以）或者ID名称；

参数2：指定是采用那种方式弹出；

参数3：如何是iframe方式，这里是设置iframe的大小；

参数4：是否出现关闭按钮，默认是显示关闭按钮，如果弹出的页面中有关闭按钮，请设置成0。

1、点击这里查看iframe方式；

function testIframe(){
	loader('ui/modal',function(){
		$app.modal.show('http://www.qq.com','iframe',{'width':'500px','height':'300px'});
	});	
}

2、点击这里查看file方式；

function testFile(){
	loader('ui/modal',function(){
		$app.modal.show('modal.php.html','file');
	});
}

3、点击这里查看ID方式；

function testId(){
	loader('ui/modal',function(){
		$app.modal.show('test','','',0);
	});
}
