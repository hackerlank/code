jslib1.4.2微博收听

只需传入需要收听或者取消收听的微博帐号微博

如果需要回调，请增加回调函数

点击收听 bondli

function followme(){
	loader('weibo/follow',function(){
		$app.follow.listen('pconline',function(e){
		    if(e.result == 0){
		        alert('温馨提示：收听官方微博成功！');
		    }
		});
	});
}

点击取消收听 dcshi

function unfollow(){
	loader('weibo/follow',function(){
		$app.follow.cancel('dc_shi');
	});
}
