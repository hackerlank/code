dialog组件

该组件和modal组件相似，但是提供了一个功能更强大的弹出提示框，多用于活动结束了告知用户领奖等说明。

测试示例：点击弹出层

支持参数配置：

title：显示在蓝色条上的文字

content：显示在白色主题区域的内容

width：显示的宽度

height：显示的高度

mask：是否遮罩，true是有遮罩层

callback：关闭时的回调函数

function open(){
	loader('dialog/dialog',function(){
		$app.dialog.show({
			title: '你好',
			content: '你好，哈哈',
			width: 400,
			height: 150,
			mask: true,
			callback: ''//function(){alert('我是回调函数!');}
		});
	});
}
