验证码组件

1、弹出验证码：$app.verifycode.show(); 点击弹出验证码

2、关闭弹出的验证码：$app.verifycode.close();

3、弹出验证码方式的回调函数中取得验证码：function callback(e){alert('我是回调函数!你刚才输入的验证码是：'+e.verifycode);}

function openVerify(){
	$app.verifycode.show('请输入验证码', callback);
}
function callback(e){
	//请在此处编写你的投票兑换等逻辑
	alert('我是回调函数!你刚才输入的验证码是：'+e.verifycode);
}

4、直接显示验证码：$app.verifycode.nchangeImg();
